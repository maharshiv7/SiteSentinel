About SiteSentinel

SiteSentinel is a Full-Stack Cyber Security reconnaissance tool. It acts as an automated security grader that inspects web servers for critical missing HTTP headers and identifies potential vulnerabilities like Clickjacking and Cross-Site Scripting (XSS).

Powered by a Python/Flask backend and an intuitive Tailwind CSS frontend, this tool also features Open-Source Intelligence (OSINT) capabilities, performing real-time WHOIS lookups to gather domain registration data.
